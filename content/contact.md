## Anonymous notes

For comments you'd rather send without a return address, the simplest path is a one-way email from a throwaway account (Proton, Tutanota, etc.). Open a new account, send to `contact@trevorc.com`, then close the account. The receiving inbox sees only the temporary address.

If you want a more durable anonymous channel set up later (a Tally form, a Formspree endpoint, a Signal username), I can add one. For now, throwaway email is the lightest option.
