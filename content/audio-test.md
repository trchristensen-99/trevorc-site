## How to swap the script

Edit `scripts/generate-tts-samples.py`. Change `SAMPLE_TEXT`. Re-run from the repo root:

```
.tts-venv/bin/python scripts/generate-tts-samples.py
```

The full Edge TTS voice list (over 300 voices in many languages) is browseable with:

```
.tts-venv/bin/edge-tts --list-voices
```

To narrow to English: pipe through `grep en-`.

## What about the full audio player features?

The samples above use plain HTML5 `<audio>` for fast comparison. The full player (save place across visits, playback speed, skip 15s, time remaining) is on any essay that sets `audio:` in its frontmatter. See [[example-essay]].
